About Website:

This website provides simple guides for complex tasks mostly in the infrastructure as a code model format. Non-routing/switching guides will be provided with a pre-configured vagrant file for the virtual machines' deployment and networking. The vagrant environment will further be set up with provisioning using bash, ansible, chef, etc. The routing/switching guides will be set up in gns3 and are more time-consuming than the rest. All the files/scripts necessary for the guides are provided on my Projects repo in GitHub.
